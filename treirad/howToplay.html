Tic Tac Toe

General information about game


Purpose: for players to place their 3 allotted marks (X or O) in a continuous line, vertically, horizontally, or diagonally.

Rules: each player chooses a mark and takes turns to place them on the grid. The game is over when a player gets a continuous line of 3, or the 9-square grid is full.

How to play: players take turns to place their marks in an attempt to get a continuous line of 3 cells.

Strategy: it is a solved game, meaning that there are strategies that can be used for the best result in each game.

Game description


Before each game person should choose names for player X and player 0 who will play next round

On the board can be only 3 steps for X and 3 steps for 0

If game didn't finish after 6 steps, player can tick on X or 0 accordingly and place move to another cell

Game will finished when X or 0 will get 3 x or 3 0 in the row, collum or diagonally

In some special cases can be pressed restart button, and game will start from the beginning, with the same players
